## Run Locally  
Clone the project  

~~~bash  
  git clone git@github.com:emanuel-ra/Frontend-Mentor-Blog-preview-card.git
~~~

Go to the project directory  

~~~bash  
  cd app
~~~

Install dependencies  

~~~bash  
pnpm install
~~~

Start the server  

~~~bash  
pnpm run dev
~~~
